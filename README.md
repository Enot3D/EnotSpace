# ⬡ ENOT SPACE — Система управления 3D-производством

PWA-приложение для управления бизнесом на 3D-принтерах.
Работает как сайт и как приложение на iPhone через Safari.

## 🆕 Новые возможности

- 🔄 **Синхронизация между устройствами** — данные автоматически синхронизируются через Firebase
- 🔐 **Система авторизации** — защищённый вход с ролями пользователей
- 👥 **Управление пользователями** — создавай аккаунты для сотрудников
- ⚡ **Быстрая смена статуса** — меняй статус заказа одной кнопкой
- 📊 **Реалтайм обновления** — изменения видны на всех устройствах мгновенно

## 👤 Роли пользователей

- **Админ** — полный доступ + управление пользователями
- **Помощник владельца** — полный доступ (без управления пользователями)
- **Менеджер** — заказы, клиенты, склад (без финансов)
- **Оператор принтера** — только заказы и принтеры

## 🔥 Настройка Firebase

Для работы синхронизации нужно настроить Firebase. Подробная инструкция: [FIREBASE_SETUP.md](./FIREBASE_SETUP.md)

Быстрый старт:
1. Создай проект на https://console.firebase.google.com/
2. Включи Authentication (Email/Password) и Firestore
3. Скопируй `src/firebase/config.example.js` в `src/firebase/config.js`
4. Вставь свою конфигурацию из Firebase Console
5. Создай первого пользователя-админа через Firebase Console

**ВАЖНО:** Файл `src/firebase/config.js` с реальными ключами не попадёт в Git (он в `.gitignore`)

## 📱 Установка на iPhone

1. Открой сайт в **Safari** на iPhone
2. Нажми кнопку **«Поделиться»** (квадрат со стрелкой вверх)
3. Выбери **«На экран «Домой»»**
4. Нажми **«Добавить»** — приложение готово ✅

## 🚀 Деплой на GitHub Pages

### 1. Создай репозиторий на github.com (Public)

### 2. Измени homepage в package.json:
```
"homepage": "https://ТВО_НИК.github.io/enot-space"
```

### 3. Установи и собери:
```bash
npm install
npm run build
```

### 4. Залей на GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ТВО_НИК/enot-space.git
git push -u origin main
```

### 5. Задеплой:
```bash
npm run deploy
```

Приложение доступно по: `https://ТВО_НИК.github.io/enot-space`

## 💻 Локальный запуск
```bash
npm install
npm start
```

## 🗄 База данных
Все данные хранятся в localStorage браузера на устройстве.
Работает офлайн. Экспорт доступен в разделе Настройки.
