# Telegram Bot для ENOT SPACE

Telegram бот для управления производством и получения уведомлений.

## Быстрый старт

### 1. Запуск бота

```bash
cd telegram-bot
node bot.js
```

Бот запустится и будет ждать сообщений.

### 2. Настройка в приложении

1. Открой бота в Telegram (найди по имени или используй ссылку)
2. Напиши `/start`
3. Скопируй свой Chat ID
4. Открой приложение ENOT SPACE → **Ещё** → **Настройки**
5. Вставь Chat ID в поле "Telegram Chat ID"
6. Сохрани

### 3. Готово!

Теперь напоминания будут приходить в Telegram автоматически.

## Команды бота

### Основные
- `/start` — приветствие и получение Chat ID
- `/help` — справка по всем командам
- `/chatid` — показать твой Chat ID

### Напоминания
- `/reminders` — список активных напоминаний
- `/add <текст>` — создать быстрое напоминание
  - Пример: `/add Позвонить клиенту через 2 часа`

### Статистика
- `/stats` — сводка по заказам и финансам
- `/printers` — статус всех принтеров
- `/finance` — финансовая сводка

## Как работают уведомления

1. Создай напоминание в приложении (Планировщик → + Напоминание)
2. Включи чекбокс **"Отправить в Telegram"**
3. Настрой время и "Напомнить заранее"
4. Сохрани

Бот автоматически отправит уведомление в нужное время!

## Формат уведомлений

```
🔔 Напоминание

📋 Позвонить клиенту
Обсудить новый заказ

⏰ 15 апреля, 14:00

🔴 СРОЧНО!
```

## Требования

- Node.js 14+ (для запуска бота)
- Интернет-соединение

## Запуск бота как сервис (опционально)

### Windows (с помощью PM2)

```bash
npm install -g pm2
pm2 start bot.js --name enot-bot
pm2 save
pm2 startup
```

### Linux (systemd)

Создай файл `/etc/systemd/system/enot-bot.service`:

```ini
[Unit]
Description=ENOT SPACE Telegram Bot
After=network.target

[Service]
Type=simple
User=your-user
WorkingDirectory=/path/to/printboss/telegram-bot
ExecStart=/usr/bin/node bot.js
Restart=always

[Install]
WantedBy=multi-user.target
```

Затем:
```bash
sudo systemctl enable enot-bot
sudo systemctl start enot-bot
```

## Troubleshooting

### Бот не отвечает
- Проверь, что `bot.js` запущен
- Проверь токен бота в файле
- Проверь интернет-соединение

### Уведомления не приходят
- Убедись, что Chat ID правильно указан в настройках
- Проверь, что чекбокс "Отправить в Telegram" включен
- Проверь, что приложение открыто (проверка каждые 5 минут)
